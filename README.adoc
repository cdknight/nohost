=== Installing with Cargo

=== Installing with Nix

You should be able to just add the following to your imports in `configuration.nix`.

[source,nix]
----
imports = [ "${builtins.fetchTarball "https://github.com/cdknight/nohost/archive/master.tar.gz"}/" ]
----
